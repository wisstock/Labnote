This collection is a multi-user shareable SnapGene resource.
You can open it by double-clicking the "Open This Collection" file.
File system permissions control which users can read and write its files.
Any changes made here will be automatically seen by other privileged SnapGene users.

Each sub-directory holds files organized into these categories:
* DNA Files
* RNA Files
* Protein Files
* Miscellaneous Files (text and images)

Files can be added and removed through SnapGene 4.0+ or with a file browser like Finder in macOS, File Explorer in Windows, or Nautilus in Linux.

Refer to the SnapGene documentation for more information.
